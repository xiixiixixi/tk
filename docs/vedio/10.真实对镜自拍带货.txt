# 强种草口播公式：
每条视频的分段口播必须至少包含以下 5 个转化成分中的 4 个：

1. 反差开场：
本来没期待 / 差点错过 / 以为普通 / 结果意外好。

2. 具体细节：
必须讲一个具体可感知的细节，比如版型、质感、触感、容量、上脸效果、使用方便程度、搭配完成度。

3. 真实动作证明：
口播必须和人物正在做的动作同步。
整理衣摆时讲版型；
靠近镜子时讲面料；
打开包时讲容量；
使用厨房用品时讲省事；
上脸时讲质地；
走路时讲脚感。

4. 生活场景代入：
必须让观众知道这个产品适合什么时候用。
例如出门前、上班前、约会前、健身后、做饭时、收纳时、旅行时、日常 errands。

5. 软转化结尾：
不能强行喊单。
要用“我会留下 / 我会常用 / 比我想象中实用 / 难怪会被问链接”这类自然表达。

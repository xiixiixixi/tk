## 四、核心创作公式

所有视频必须遵守以下成交公式：

真实人物出镜  
→ 真实痛点吐槽  
→ 痛点细节放大  
→ 产品自然出现  
→ 产品实测解决  
→ 证明动作  
→ 结果展示  
→ 人物真实反应  
→ 自然种草  
→ 指引下单

视频不能像广告。  
不能一上来拍产品。  
不能一上来开箱。  
不能一上来讲卖点。  
必须先让用户觉得：“这就是我遇到的问题。”

## 八、爆款结构库

每条视频都必须从下面结构中选择一种，并在【视频策略】里写明。

不要所有视频都用同一种结构。

### 结构1：痛点吐槽型
人物吐槽痛点 → 展示问题 → 产品解决 → 结果 → 下单指引

### 结构2：旧方法失败型
人物说之前一直用错方法 → 展示旧方法无效 → 产品出现 → 实测 → 结果 → 下单指引

### 结构3：评论区问题型
人物说“有人问这个怎么解决” → 展示问题 → 产品出现 → 实测 → 答案 → 下单指引

### 结构4：懒人解决型
人物说“不想再花时间弄这个” → 展示麻烦 → 产品出现 → 快速解决 → 轻松结果 → 下单指引

### 结构5：真实测试型
人物说“我想看看它到底有没有用” → 展示问题 → 产品出现 → 全程实测 → 结果 → 下单指引

### 结构6：反差惊喜型
人物说“我以为这个没必要” → 展示痛点 → 产品出现 → 使用后反应 → 结果 → 下单指引

### 结构7：尴尬场景型
人物讲一个真实尴尬场景 → 展示问题 → 产品解决 → 结果 → 下单指引

### 结构8：省力对比型
人物说旧方法太费劲 → 展示旧方法麻烦 → 产品出现 → 轻松解决 → 结果 → 下单指引

### 结构9：Satisfying 实测型
人物讲痛点 → 展示脏乱细节 → 产品实测 → 过程很解压 → 结果 → 下单指引

### 结构10：小东西大作用型
人物说“这个小东西我本来没期待” → 展示痛点 → 产品出现 → 解决问题 → 惊喜反应 → 下单指引

## 九、证明动作硬规则

每条视频必须有至少 1 个“证明动作”，最好放在镜头4或镜头5。

证明动作是让观众相信产品真的有用的关键。

不同产品可选证明动作：

清洁类：
- 刷之前手指擦一下脏缝隙
- 刷后再用手指擦一下变干净的区域
- 纸巾擦拭前后对比
- 刮水条把脏水推向地漏
- 半边清洁 / 半边未清洁对比
- 低角度拍水渍被推走
- 墙角、缝隙、边缘重点清洁

收纳类：
- 凌乱前后对比
- 一只手拿出物品证明更方便
- 抽屉 / 柜子打开前后对比
- 原本堆在桌面的东西被整齐放好
- 展示节省空间

厨房类：
- 旧方法麻烦对比
- 使用前后时间感对比
- 食材 / 厨具 / 水槽状态变化
- 清理前后对比
- 直接展示省步骤

美容个护类：
- 使用步骤近景
- 使用前后状态对比
- 真实表情反馈
- 手部 / 头发 / 皮肤表面状态展示
- 注意不能夸大医学效果

车品类：
- 安装前后对比
- 车内凌乱变整齐
- 单手使用证明便利
- 通勤场景实测

宠物类：
- 毛发 / 食物 / 玩具凌乱前后对比
- 宠物自然入镜
- 主人使用后轻松反应
- 注意不能表现危险动作

证明动作必须真实可信。  
禁止夸张到不真实的瞬间奇迹效果。
